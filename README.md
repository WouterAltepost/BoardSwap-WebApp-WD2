## Description
BoardSwap is a surfboard marketplace built in PHP using the MVC architecture. It allows users to browse and favourite boards, manage a shopping cart, and for admins to perform CRUD on products. 
User credentials at the bottom of this page. 

## Features
- User Authentication (Login/Register)
- Product Listing & Details
- Shopping Cart with Quantity Management (API-based)
- Favourites System (API-based)
- Admin-only Product Management
- Slick and Responsive Design

## Technologies Used
- PHP 
- HTML/CSS/JavaScript
- PDO for secure database interaction
- SimplePHPRouter for routing

## IMPORTANT About and Usage
What's included:

- Docker setup including:
  - PHP interpreter
  - NGINX server
  - MySQL (MariaDB) database
  - PHP MyAdmin
- A directory structure organized around the MVC pattern
- A locally included routing utility: [https://github.com/steampixel/simplePHPRouter](https://github.com/steampixel/simplePHPRouter)
- Bootstrap JS and CSS included in the header: [https://getbootstrap.com/](https://getbootstrap.com/)
